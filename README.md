# Mini Canva - Text Editor

A lightweight Flutter app that mimics **Canva's text editing** features. Add, move, style, and edit text on a canvas with **undo/redo** support.

---

## Features
- **Add Text**  
- **Drag to Move**  
- **Edit Text Content**  
- **Change Font Size**  
- **Bold / Italic Toggle**  
- **Delete Selected Text**  
- **Undo & Redo**  
- **Visual Selection Border**

---
